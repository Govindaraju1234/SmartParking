# 🚗 Smart Parking Management System

A web-based Smart Parking Management System developed using Spring Boot, Thymeleaf, MySQL, and Bootstrap. The system allows users to book parking slots, make payments, generate QR codes, and manage parking history through a user-friendly interface.

## 📌 Features

- User Registration and Login
- Admin Dashboard
- User Dashboard
- Parking Slot Management
- Book Parking Slot
- Online Payment
- QR Code Generation
- Booking Management
- Payment History
- Parking History
- Revenue Dashboard
- Responsive Bootstrap UI

## 🛠️ Technologies Used

- Java 21
- Spring Boot 3.5
- Spring MVC
- Spring Data JPA
- Thymeleaf
- MySQL
- Bootstrap 5
- HTML
- CSS
- JavaScript
- Maven
- ZXing QR Code Library

## ⚙️ Installation

1. Clone the repository

bash
git clone https://github.com/Govindaraju1234/Smart-Parking-Management-System.git


2. Open the project in VS Code or Eclipse.

3. Configure MySQL in application.properties.

4. Run the project:

bash
mvn spring-boot:run


5. Open:


http://localhost:8080
